# CollabEditor
Real-time P2P Multiplayer for the Geometry Dash 2.2081 Editor.

## 🚀 Quick Features
- **Native UI**: Standard GD "Sandwich" buttons in the Level Info layer.
- **P2P Sync**: Real-time object placement and synchronization (WIP).
- **Session Control**: Host and join collab sessions via Account IDs.
- **Auto-Save**: Integrated CollabManager for persistent session data.

## 🛠️ Build Requirements
- Geode SDK (v5.5.3+)
- C++23 / Clang
- i3/Ryzen Optimized: Uses compile.bat for low-capacity CPU builds.

## 📂 How to Build
```powershell
git clone https://github.com/Ender68-ai/Editor-collab.git
cd Editor-collab
.\compile.bat
```
